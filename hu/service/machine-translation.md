# Gépi fordítás

## Bevezetés

Főleg az ingyenes, mindenki részere hozzáférhető alternatívák az érdekesek.

Ideális, ha nem épít profil a tevékenységünkből, illetve ha API segítségével beköthető akár egy böngészőbe vagy csevegőprogramba.
